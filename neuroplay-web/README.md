# NeuroPlayPro Web (без сборки)

Веб-интерфейс для NeuroPlayPro API (`http://127.0.0.1:2336`) на чистом HTML/CSS/JS.

## Запуск
1. Запустить NeuroPlayPro.
2. Открыть `neuroplay-web/index.html` в браузере.
3. Нажать:
   - `Тест` — проверка API
   - `Поиск` / `Устройства` — обнаружение девайсов
   - `Подключить` → `Запуск` устройства
   - `Запись` / `Пауза`

## Реализовано

## 1) Усреднение (4 режима)
- Индивидуальные (6 каналов)
- Попарное: (Fp1+Fp2)/2, (T3+T4)/2, (O1+O2)/2
- Полушария: (Fp1+T3+O1)/3, (Fp2+T4+O2)/3
- Глобальное: (все 6)/6

## 2) Визуализации (6 подвкладок)
- Ритмы (кольца)
- Спектр (тепловой overlay)
- Пиковая частота (цвета по доминирующей частоте)
- Связность (линии по матрице корреляций)
- Энтропия (Shannon)
- Статистика (панель + tooltip)

## 3) Матрица корреляций 6×6
- Реальный расчет из `lastSpectrum`
- Переключатель band (delta/theta/alpha/beta/gamma)
- Цветовая кодировка от -1 до +1
- Экспорт CSV

## 4) Статистика
- Mean, StdDev, Min/Max
- Skewness, Kurtosis
- CV
- Normality (эвристика)
- Percentiles (Q25/Q50/Q75), IQR
- Распределение мощностей по ритмам

## 5) Нормализация
- Min-Max
- Z-score
- By-band normalization
- Baseline subtraction
- Relative %

## 6) Запись
- Start/Stop/Pause/Continue
- Декодирование EDF (base64) и скачивание
- JSON metadata

## 7) Подключение и устройство
- Статусы подключения (disconnected/connecting/connected/searching/error)
- Поиск/список устройств
- Start/Stop device
- `currentDeviceInfo` + `quality` + фильтры в UI

## 8) Мультиоконность
- BroadcastChannel синхронизация:
  - вкладки
  - UI состояния подключения
  - UI состояния записи
- Базовый master/slave флаг

## Важные замечания
- API должен быть доступен по `http://127.0.0.1:2336`.
- Канальный порядок для 6ch: `O1, T3, Fp1, Fp2, T4, O2`.
- Проект реализован без тестирования по прямому требованию.
